# Spawn Checklist

Use this checklist when creating a new Fox Mountain app from the template.

## Rename / Identity
- [ ] App name updated
- [ ] Scheme updated
- [ ] Bundle identifier updated
- [ ] Product name updated
- [ ] Display name updated

## Product Surface
- [ ] Replace placeholder home screen
- [ ] Update settings/about content
- [ ] Add app-specific models/view models/services
- [ ] Replace placeholder icon assets

## Monetization / Analytics
- [ ] RevenueCat config reviewed
- [ ] AdMob config reviewed
- [ ] Firebase / analytics config reviewed
- [ ] Privacy / ATT implications reviewed

## App Store / Deployment
- [ ] Appfile updated
- [ ] Deliverfile updated
- [ ] Matchfile reviewed
- [ ] App Store Connect app record created
- [ ] Screenshots generated
- [ ] Privacy labels prepared

## Verification
- [ ] `xcodegen generate`
- [ ] local build passes
- [ ] TestFlight lane reviewed
